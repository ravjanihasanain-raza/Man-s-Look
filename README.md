# Man's Look – E-Commerce Store

## Overview

Man's Look is a full-featured e-commerce web application developed using PHP and MySQL. The platform enables customers to browse products, manage their shopping cart, create accounts, and place orders through a responsive and user-friendly interface.

The system includes two primary modules: **User** and **Admin**, providing a complete online shopping and product management experience.

## Installation

### Requirements

* PHP 8+
* MySQL
* XAMPP / WAMP

### Setup Steps

1. Clone the repository.
2. Copy the project folder into the `htdocs` directory.
3. Start Apache and MySQL services.
4. Import the database file into phpMyAdmin.
5. Configure the database connection settings.
6. Open the project in your browser.

Example:

```bash
http://localhost/mans-look
```
